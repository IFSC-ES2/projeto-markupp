# ADR-0004: Aplicação web

## Contexto

O usuário precisa de uma interface para interagir com seus documentos

## Decisão

Aplicação web acessível via navegador. Frontend ainda em definição entre React com Vite ou Ruby on Rails

## Alternativas consideradas

- Electron/Tauri: mais complexo de distribuir, sem ganho para o MVP
- Mobile nativo: fora do escopo

## Consequências

- Acessível de qualquer dispositivo com navegador
