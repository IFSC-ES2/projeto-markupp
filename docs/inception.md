# Inception - Markupp

## Problema

Anotações pessoais e conteúdos gerados por agentes de IA ficam espalhados em ferramentas diferentes, sem um lugar central para reunir tudo. Falta um "segundo cérebro" que funcione como ponto único de acesso tanto para o usuário quanto para seus agentes.

## Usuários e interessados

- Pessoas que usam agentes de IA no dia a dia e precisam de um lugar para reunir suas anotações e as saídas geradas pelos agentes
- Qualquer usuário que queira um segundo cérebro self-hosted em markdown

## Proposta de valor

Um segundo cérebro centralizado e self-hosted onde o usuário e seus agentes de IA compartilham a mesma base de anotações em markdown, acessível via interface web e API REST.

## Objetivos do semestre

- Entregar o MVP funcional: CRUD completo de arquivos markdown com interface web
- Estabelecer a arquitetura base (backend, frontend, armazenamento)

## Premissas, restrições e limitações

- Equipe de 4 pessoas com papéis definidos (arquiteto, DevOps, QA, Scrum Master)
- O sistema deve ser self-hosted, sem dependência de serviços terceiros para funcionamento
